```
200 - 299 接口处理成功，对应不同的成功信息


# 接口更新代码 303->307->301
301 记录已被永久转移到新的资源上（接口更新，旧接口已不再使用，本次请求已不再返回数据）
302 记录被临时转移（请使用返回的附带参数重新请求）
303 资源应该可以在另一个接口上获得（通常出现在接口即将更新，新接口已包含当前接口的数据）
307 接口已更新，当前接口仍能正常返回信息（同200）但应尽快根据变更通知更新接口

304 资源未被修改（基于所携带的上次所获得资源的时间，请使用缓存）

# 业务参数检查
340 - 349 参数为空
350 - 359 参数错误

# 请求资源检查错误代码
401 未携带 access_token
403 access_token 错误
404 请求的记录不存在（出现在获取视频，轮播等信息时）
405 请求方法错误（部分接口仅支持`POST`）
407 非白名单IP（暂未启用）
408 处理超时，请尝试重试
409 所更新内容存在冲突无法处理
410 请求资源被移除
413 所设置资源的值超过限制（比如轮播内视频数量）
414 URL长度超过限制（可能出现在回调链接超过长度限制等）
415 所请求媒体格式不支持（可能出现在照片或视频上传中）
416 可能会出现在分页数据，超过最大页数限制
421 服务器无法产生响应（多出现在内部DB或Cache链接出现问题）
423 资源被锁定无法获取或更改
424 之前的错误导致本次错误，请不要再重试
426 当前接口强制要求使用`HTTPS`(正式环境已全面启用https,建议部署环境使用https)
428 此代码出现在为保证接口并发时所下发给上次获取的内容已被更新，而导致当前更新操作失败（版本号不是最新）
429 请求频率超限
451 因法律原因资源被限制（视频或频道违规被禁）

# 业务处理错误代码
500 数据处理失败，未知错误
540 - 549 参数处理失败
550 - 559 缓存（读/存）失败
560 - 569 数据库（读/存）失败
600 - 699 第三方接口错误
```
